# Drag-Drop-File-Uploader
This project is a simple and interactive drag-and-drop file uploader built with HTML, CSS, and JavaScript.
It supports image uploading with preview and includes a progress bar animation to simulate the upload process.

1- Features
- Drag and drop upload area
- Image preview after uploading
- Simulated progress bar
- Validates image file types (JPG, PNG, GIF)
- Error message for invalid file types
- Clean and responsive UI

2- How It Works
- Drag an image into the upload box or select a file manually.
- The file is validated to ensure it is an image.
- A preview of the selected image is shown.
- A progress bar runs to simulate an upload.
- Once completed, only the uploaded image section remains visible.

3- File Structure
Drag & drop file uploader/
│── index.html        
│── style.css        
│── script.js         
└── README.md         

4- Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

5- How to Run
- Download or clone the repository
- Open index.html in any browser.
- Start uploading images using drag-and-drop or by selecting a file.
